# 🥪 Сэндвич-шопы — Фронтенд

Фронтенд-приложение для национальной сети сэндвич-шопов с функцией онлайн-заказов и доставки.

## 📋 Функционал

### Главная страница (Preauthorize)
- Преимущества сети
- Популярные сэндвичи (3 позиции)
- Промокод для новых пользователей (WELCOME30 — скидка 30%)
- Информация для партнёров (франшиза)

### Страница меню
- Просмотр всех сэндвичей (10 позиций)
- Фильтрация по категориям (Куриные, Мясные, Вегетарианские, Рыбные)
- Поиск по названию
- Сортировка по цене
- Переключатель вида: сетка / список
- Пагинация (6 товаров на странице)
- Sidebar с популярным и акциями

### Карточка товара (модальное окно)
- Основная информация (название, цена, статус доступности)
- Галерея изображений с навигацией
- Состав и пищевая ценность (БЖУ)
- Отзывы и рейтинг
- Добавление в корзину

## 🚀 Быстрый старт на Windows

### Требования
- **Node.js** версии 18.x или выше
- **npm** (идёт в комплекте с Node.js)

### Шаг 1: Установка Node.js

1. Перейдите на официальный сайт: https://nodejs.org/
2. Скачайте LTS-версию для Windows
3. Запустите установщик и следуйте инструкциям
4. После установки перезапустите терминал

### Шаг 2: Проверка установки

Откройте PowerShell или Command Prompt и выполните:

```bash
node --version
npm --version
```

Должны отобразиться версии (например, `v20.11.0` и `10.2.4`)

## 🎯 Автоматический запуск (скрипты)

### Windows (.bat)
Дважды кликните на файл `start.bat` или выполните в терминале:

```bash
start.bat
```

### Windows (PowerShell)
Откройте PowerShell и выполните:

```powershell
.\start.ps1
```

> ⚠️ Если скрипт не выполняется, разрешите выполнение:
> ```powershell
> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
> ```

### macOS / Linux (.sh)
Откройте терминал и выполните:

```bash
chmod +x start.sh
./start.sh
```

## 📝 Ручной запуск

### Шаг 3: Установка зависимостей

```bash
cd sandwich-shops-front
npm install
```

### Шаг 4: Запуск приложения

```bash
npm run dev
```

После запуска в терминале появится ссылка:
```
  VITE v5.0.8  ready in 500 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

### Шаг 5: Открытие в браузере

Перейдите по ссылке `http://localhost:5173/`
